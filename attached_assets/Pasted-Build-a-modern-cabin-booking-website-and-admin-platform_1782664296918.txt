Build a modern cabin booking website and admin platform for “Walker Creek Farms & Cabins.”

Goal:
Replace the current Smoobu booking setup with our own platform. The current website has a rustic wood/cabin theme, cream content cards, dark brown accents, nature/cabin photos, and simple navigation. Keep that warm rustic Appalachian cabin feel, but modernize it so it feels professional, mobile-friendly, clean, and trustworthy.

Tech stack:
- Next.js or React/Vite
- TypeScript
- Tailwind CSS
- Supabase for database/auth/storage
- Stripe for payments
- Resend or SMTP for email notifications
- Deployable to Vercel or Netlify
- Include a good README with setup steps

Public website pages:
1. Home
   - Hero section with cabin/nature imagery
   - “Walker Creek Farms & Cabins”
   - Rustic but modern design
   - Short intro about secluded cabins/yurts in West Virginia
   - Featured cabins
   - Book Now CTA

2. Cabins / Yurts
   - List of rentals with photos, description, amenities, guest count, bedrooms, bathrooms, nightly price
   - Each rental has its own detail page
   - Photo gallery
   - Availability calendar
   - Book Now button

3. Food
   - Menu/information page
   - Keep ability to list partner food options, prices, order instructions, and contact info

4. FAQs
   - Location
   - Booking/cancellation policy
   - What is provided
   - Vehicle access
   - TV/Wi-Fi
   - Hot tubs/fire pits/etc.

5. Gift Certificates
   - Explain how gift certificates work
   - Allow purchase through Stripe
   - Admin can create/edit gift certificate options

6. Contact / Booking
   - Contact info
   - Embedded booking CTA
   - Contact form

7. Blog
   - Simple blog list and post pages
   - Admin can create/edit/delete posts

Booking system:
- Users choose rental, dates, guest count, and submit booking
- Show unavailable dates clearly
- Prevent double booking
- Pricing by night
- Weekend/weekday pricing support
- Cleaning fee/taxes optional
- Stripe checkout for payment
- Booking confirmation email to guest
- New booking notification email to owner
- Admin can manually add bookings for phone bookings
- Admin can block dates manually
- Admin can cancel/refund-mark bookings

Admin dashboard:
- Secure login
- Manage rentals
- Manage photos
- Manage amenities
- Manage rates
- Manage bookings
- Manage blocked dates
- Manage food page content
- Manage FAQ content
- Manage gift certificates
- Manage blog posts
- View upcoming arrivals and recent bookings
